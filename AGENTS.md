# 犯罪意图识别框架

这是一个机器学习任务，旨在识别互联网中潜在的犯罪意图。该任务涉及自然语言处理、模型微调和偏好优化等技术。

## 环境
- 需要跑代码时，使用 conda 环境，名称 ML 
- 环境 ML 已经建好，有一些基础的包，激活即可
- 向 ML 环境中安装包时，必须使用 `pip install` 并使用清华源加速，不使用 `conda install`，以确保包的兼容性和稳定性
- 如果遇到难以解决的依赖冲突，新建 conda 环境，统一使用 `pip install` 不要混用 `conda install` 和 `pip install`，以避免依赖冲突

## 写代码前的工作
- 查阅 docs/DevelopLog.md ，了解之前的开发内容
- 同时查看 git 提交记录，了解代码的修改历史

## 编码注意事项
- 禁止一切 fallback 和补丁打补丁， 代码必须是干净的、可维护的
- 每次要做修改都果断，不对之前出错的逻辑进行兼容
